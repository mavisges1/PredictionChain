# PredictionChain

Decentralized prediction market built on Ethereum Sepolia testnet.

## Contracts
- RewardToken.sol (ERC-20 participation token)
- PredictionMarket.sol (prediction logic)

## Features
- Create prediction events
- Place bets with test ETH
- ERC-20 reward token minting
- Finalize events and claim rewards
- MetaMask integration

## Network
- Sepolia Testnet
- Deployed using Remix + MetaMask

## Note
This project is developed for educational purposes only.
